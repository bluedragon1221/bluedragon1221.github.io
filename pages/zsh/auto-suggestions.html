Auto Suggestions

I use auto suggestions in my ZSH prompt. Auto Suggestions use my zsh history to predict what I will type next. They come in handy when I need to retype a long command or use a short command many times.

Installation

The installation is very similar to Syntax Highlighting.

Clone the git repository,

cd ~/.config/zsh
git clone https://github.com/zsh-users/zsh-auto-suggestions

and source it in your ~/.zshrc.

### ~/.zshrc ###
source ~/.config/zsh/zsh-autosuggestions/
