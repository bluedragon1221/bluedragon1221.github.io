ZSH Prompt

zsh completely fixes the bash PS1 variable with prompt expansions. Since the default zsh prompt kind of sucks, lets make it better.

For a more default-looking prompt, try this in your ~/.zshrc. This will show the current working directory, then a %.

### ~/.zshrc ###
PROMPT="%~ %% "

Using more prompt expansions, this is my zsh prompt.

### ~/.zshrc ###
PROMPT="%F{blue}%~%f %B%(?.%F{green}.%F{red})%f%b "

This is the list of all prompt expansions.

If this level of customization isn't enough, I suggest you check out starship.
