Syntax Highlighting

zsh-syntax-highlighting is a plugin that colors invalid commands red and underlines valid paths.

Installation

Clone the zsh-syntax-highlighting git repository,

mkdir ~/.config/zsh; cd ~/.config/zsh 
git clone https://github.com/zsh-users/zsh-syntax-highlighting

then source it at the end of your ~/.zshrc.

## ~/.zshrc ##
...
source ~/.config/zsh/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh

For more info on zsh-syntax-highlighting, see the git repo.
