Aliases

Aliases are awesome! They allow you to shorten any any command that you type frequently. I talk about aliases a lot, so I won't go into any specific usecases.

Create an Alias File

I have so many aliases that I keep them in a separate file and source that file.

create the file,

touch ~/.config/zsh/aliasrc

and source the file in your ~/.zshrc.

### ~/.zshrc
source ~/.config/zsh/aliasrc

My Alias file

This is my aliasrc. You can use it for insiration in your own aliasrc.

Remotely Add Aliases

I have a bash function that will remotely add an alias to my aliasrc.

add_alias() {
echo $1 >> ~/.config/zsh/aliasrc
}
